# Dice: Stremio Addon

Built with [stremio-addon-sdk](https://github.com/Stremio/stremio-addon-sdk)

## Install

- Install nodes packages: `npm install`
- Create a .env file at the root, add and update this variables:
  
```
ENV="dev"
PORT={PORT}
DOMAIN="{DOMAIN_URL}"
CINEMETA="{CINEMETA_URL}"
```
